# Приглашение Виктории — Google Sheets + уведомление на почту

Этот вариант предназначен для GitHub Pages. После нажатия **«Подтвердить ❤️»**:

1. новая строка появляется в Google Таблице;
2. письмо отправляется на `artemcepko69@gmail.com`;
3. на сайте показываются конфетти и экран благодарности.

Сервер Vercel, Telegram-бот, Node.js и `package.json` больше не нужны.

## Шаг 1. Создайте Google Таблицу

1. Создайте новую Google Таблицу, например **«Ответ Виктории»**.
2. В таблице откройте **Расширения → Apps Script**.
3. Удалите стандартный код.
4. Скопируйте туда всё содержимое файла:

```text
google-apps-script/Code.gs
```

5. Нажмите **Сохранить**.

Лист **«Ответы»** и заголовки таблицы создадутся автоматически после первого подтверждения.

## Шаг 2. Опубликуйте Apps Script как веб-приложение

1. В редакторе Apps Script нажмите **Deploy / Начать развертывание → New deployment / Новое развертывание**.
2. В типе развертывания выберите **Web app / Веб-приложение**.
3. Укажите:
   - **Execute as / Выполнять от имени:** Me / Я;
   - **Who has access / У кого есть доступ:** Anyone / Все.
4. Нажмите **Deploy / Развернуть**.
5. Разрешите доступ к Google Таблицам и отправке писем.
6. Скопируйте адрес **Web app URL**, который заканчивается на `/exec`.

Важно: используйте именно URL развертывания `/exec`, а не ссылку редактора Apps Script.

## Шаг 3. Вставьте URL в сайт

Откройте `index.html` и найдите строку:

```js
const APPS_SCRIPT_URL = "PASTE_GOOGLE_APPS_SCRIPT_WEB_APP_URL_HERE";
```

Замените её, например:

```js
const APPS_SCRIPT_URL = "https://script.google.com/macros/s/ВАШ_ID/exec";
```

Сохраните файл и загрузите обновлённый `index.html` в репозиторий GitHub `vika`.

## Шаг 4. Проверьте

1. Подождите 1–2 минуты, пока GitHub Pages обновится.
2. Откройте сайт.
3. Выберите время и кухню.
4. Нажмите **«Подтвердить ❤️»**.
5. Проверьте:
   - лист **«Ответы»** в Google Таблице;
   - входящие и папку «Спам» на `artemcepko69@gmail.com`.

## Если нужно изменить адрес почты

В `Code.gs` измените:

```js
const NOTIFICATION_EMAIL = 'artemcepko69@gmail.com';
```

После изменения Apps Script нужно заново опубликовать:

**Deploy → Manage deployments → Edit → New version → Deploy.**

## Важно

- Сам сайт остаётся на GitHub Pages.
- Apps Script выполняет роль небольшого бесплатного сервера.
- Не публикуйте доступ на редактирование самой Google Таблицы.
- URL веб-приложения можно хранить в `index.html`: он позволяет отправить ответ, но не даёт посетителю доступ к редактированию таблицы.
